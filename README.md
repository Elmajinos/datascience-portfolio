# Machine Learning Projects

This repository contains three machine learning projects completed as part of the M2 Data Science program:

## Projects Overview

### 1. [Unsupervised Learning](./01_unsupervised_learning/)
- K-means clustering and GMM
- PCA and t-SNE dimensionality reduction
- Temperature data analysis and digit classification
- [Details](./01_unsupervised_learning/README.md)

### 2. [ECoG Regression](./02_regression_ecog/)
- Prediction of thumb flexion from brain signals
- Linear models (Least Squares, Ridge, Lasso)
- Non-linear models (SVR, Random Forest, MLP)
- [Details](./02_regression_ecog/README.md)

### 3. [Supervised Classification](./03_classification/)
- Medical diagnosis (Pima Diabetes dataset)
- Handwritten digit recognition
- Linear and non-linear classifiers
- [Details](./03_classification/README.md)

## Installation

git clone https://github.com/[your-username]/machine-learning-projects.git
